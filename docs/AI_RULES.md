# AI Rules

Before writing code:

- Read AI_CONTEXT.md
- Follow DESIGN_SYSTEM.md
- Follow ARCHITECTURE.md

Never:

- Redesign UI
- Rename packages
- Move folders
- Change architecture
- Add libraries without permission
- Create unnecessary abstractions

Always:

- Keep code simple
- Use Material 3 Expressive
- Keep Forest Green as primary color
- Maintain floating card design
- Keep Android responsible for application logic only
- Keep Unity responsible for gameplay only

If information is missing,

ask instead of assuming.
